# Mi Diario de Aprendizaje: IA y Tecnología

Este repositorio es mi bitácora personal de aprendizaje. 
Mi objetivo es documentar mi camino desde cero en el mundo de la **Inteligencia Artificial**, 
registrando cada concepto, comando y herramienta que vaya descubriendo.

Como principiante, quiero conocer los siguientes pilares:

*   **IA Local:** Aprender a ejecutar modelos de lenguaje (LLMs) en mi propio equipo. 
    *   *Hardware actual:* Macbook M3, 24 GB de RAM, 512 GB SSD.
    *   *Enfoque:* Rendimiento y configuración, como lo que más, porque sé que el equipo probablemente se quede algo justo.
*   **Prompt Engineering:** Aprender el arte de hablarle a las máquinas para obtener los mejores resultados posibles.
*   **Dominio de Comandos:** Familiarizarme con la terminal y comandos esenciales para la gestión de sistemas y herramientas de IA.
*   **Conceptos Generales:** Entender cómo funciona la IA detrás de escena (LLMs, redes neuronales, procesamiento de datos, etc.).

---

## Hoja de Ruta (Roadmap)

*A medida que avance, iré actualizando este apartado:*

- **Nivel 1: Fundamentos** (Conceptos básicos, qué es un LLM, cómo funcionan los prompts).
- **Nivel 2: Terminal y Comandos** (Instalación de entornos, gestión de carpetas, comandos básicos, principalmente de Linux / macOS).
- **Nivel 3: IA Local** (Uso de herramientas como Ollama, LM Studio, configuración de hardware o lo que considere en el momento que toque).
- **Nivel 4: Técnicas Avanzadas** (Prompting complejo, RAG, automatizaciones sencillas).

> *Nota: Creo que el Nivel 3 y el Nivel 4 serán los que más me cuesten entender.*

## ¿Cómo usar este repositorio?

Cada vez que aprenda algo nuevo, crearé una entrada en este diario con:

1.  **Fecha**
2.  **Concepto aprendido**
3.  **Ejemplos prácticos** (comandos, prompts probados, errores cometidos).
4.  **Recursos útiles** (artículos o documentación).
